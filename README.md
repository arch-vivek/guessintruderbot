# Guess The Intruder – Telegram Bot

A fully native Telegram multiplayer game where players spot the odd item from a set.

## Features

- Quick Play, Group Battle, Ranked, Duel, Daily Challenge, Endless Mode
- Inline mode (play in any chat)
- XP, levels, streaks, achievements, leaderboard
- Old-money profile card with user avatar
- Admin tools

## Setup

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with your `BOT_TOKEN` and `ADMIN_IDS`
4. Run `python main.py`
5. Set commands with BotFather (see `commands.txt`)
